# Heart_predicition

This project predicts the likelihood of heart disease in a person based on various medical attributes such as age, blood pressure, cholesterol, etc.
It uses a Deep Learning model (ANN) built with TensorFlow/Keras and provides an easy-to-use Streamlit web app for prediction.
